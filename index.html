Map Comparison of Listed Bus Stops in Albuquerque, NM

This map shows the differences betweens bus stops in Albuquerque listed within OpenStreetMap and the City of Albuquerque's public website.

This map was generated using QGIS 2.10. OpenStreetMap data was harvested using the QuickOSM feature. Common geoprocessing tools such as clips and buffers were also used.

  • See the map: Bus Stops in Albuquerque
  • Data Source: City of Albuquerque Transit Routes and Bus Stops GIS

Famous Kentuckians and Top Kentucky Tourist Destinations Map

This map highlights both famous folks from the Bluegrass and top Kentucky tourist destinations.

This map was generated using QGIS 2.10 and CartoDB. Addresses were compiled through various sources and geocoded through the MMQGIS plug-in. Both point data and the choropleth maps were altered utilizing CSS and HTML within CartoDB. Ratios were compiled through the QGIS field calculator.

  • See the map: Kentucky Proud Map
  • Data Source: Wikipedia: List of People from Kentucky

U.S. Electric Power Generation Proportional Symbol Map

This map displays electric power generation for wind, hydro, solar, and nuclear power plants in the United States.

This map was generated using Leaflet. Clicking a location on the map highlights a 500 kilometer buffer around the click point and displays only power plants within that buffer. Clicking on a proportional symbol representing each power plant displays a pop-up window listing the name of the power plant, the amount of power generated, and its distance from the click point.

  • See the map: U.S. Electric Power Generation Proportional Symbol Map
  • Data Source: U.S. Energy Information Administration

Ethiopia MAgIC Interactive Choropleth Map

This map displays a drought exposure measurement developed by GlobalAgRisk, Inc. for the country Ethiopia.

This map was generated using Leaflet. Data processing was accomplished using QGIS 2.12 and RStudio. Clicking each administrative region reveals a pop-up window displaying the month's MAgIC drought exposure proportion. Hovering over each region reveals an additional information window diplaying the entire year's MAgIC proportions. The slider tool allows the user to scroll through each month of the year as the map dynamically updates.

  • See the map: Ethiopia MAgIC Interactive Choropleth Map
  • Data Source: GlobalAgRisk, Inc.

Reference Map for Administrative Units in Peru

This is an example from a series of reference maps created for an ongoing research project in 39 low and middle income countries showing level 1 administrative units (regions/provinces).

This map was generated using QGIS 2.14. Shapefiles were provided by NaturalEarthData and were accessed via PostGIS. Polygons were stylized with the TopoColours plug-in to enhance map readability. GID and country labels were manually manipulated. A map inset was utilized to display geographically small areas. All of the 39 country maps were stylized the same way.

  • See the map: Peru Reference Map
  • Data Source: Natural Earth Data
